Colour Palette

This colour palette was designed with accessibility in mind, it was ran against multiple simulations of colour blindness to determine if it would still be indistingushable

Accessibility

These colours have been simulated against every form of colour blindness and are systematically concluded to be accessible.

How to use these colours

It is good practice to use #011627 as the primary border colour and to add depth to the page.

#fdffc is simply a off white used as whitespace on the page.

#f6c461 highlights beautifully with the primary colour and is a great secondary colour.

#54cec3 and #e71d36 are compliment colours and can be used alongside the other colours when necessary or if applicable.
