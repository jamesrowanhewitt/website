Living style guides

What is a living style guide

In programming the documentation process is often the weakest strength that programmers have, yet its one of the most important steps in programming.

Source - SmashingMagazine

Why is a living style guide useful to developers and designers

Living style guides allow front end developers to mark up code into well formulated libraries, such as established colour schemes and buttons. This aids in the gap between developers and designers keeping them in sync.

Source - SmashingMagzine

My list of living style guide tips

  • Don't be pedantic, allow for some slack
  • Doesn't have to be perfect, we just need a guide to have a consistent website
  • Essential elements: overview, rational for work, information about colour palette and logos.
  • Formulate a easy to read guide, preferbly with infographics or photos.
