Typeface

Font styles and buttons

Fonts

font: 15px/1.5 Arial, Helvetica,sans-serif;

Fonts chosen

  • Arial
  • Helvetica
  • Sans-Serif

Explanation

We have chosen these 3 fonts as they are compatible on all browsers which helps with accessiblity.

Arial must be used for all titles.

Helventica is a content font.

Sans-serif is a spare font and can be used to describe things in smaller cards.

Buttons

Buttons must follow the criteria:

  1. Highlight when mouse hovers over to aid accessiblity
  2. Stay highlighted when clicked - this is important to help with navigation and accessibility
